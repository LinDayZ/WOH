Textures are 2048 X 2048
"Installation Poser"
Just Extract to your Runtime folder models will be located in /Props/Poisen folder. 

Not tested in DAZ Studio but should be compatible

Installation .OBJ
Use "Import" functions of whatever software application you are using.

Hints and Tips 
models color tint and tone are also easily adjusted using posers "color picker" tool for even more model varience.
Also different materials work well.

with Vue import on version 8 Global highligh intensity may be too high just turn it all the way down for a matte finish, it is located in you material Tab.
